# 策略模式

定义：定义一系列的算法，把它们一个个封装起来，并且使它们可以相互替换。

以计算年终奖为例：
```
// 未使用策略模式的代码：
var calculateBonus = function(level, salary) {
    if (level === 'S') {
        return salary * 4
    }
    if (level === 'A') {
        return salary * 3
    }
    if (level === 'B') {
        return salary * 2
    }
}
```

```
// 使用策略模式的代码：
var strategies = {
    'S': function(salary) {
        return salary * 4
    },
    'A': function(salary) {
        return salary * 3
    },
    'B': function(salary) {
        return salary * 2
    }
}
var calculateBonus = function(level, salary) {
    return strategies[level](salary)
}
```

策略模式适合用来做表单校验。
